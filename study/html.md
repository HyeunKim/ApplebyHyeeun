# html.md

## html

- 웹페이지는 <html> 태그로 시작해서 </html> 태그로 끝이난다.
- 태그들로 구성
- 구성

1. head : 웹 페이지의 정보 및 문서에 사용할 외부 파일 링크 삽입 ex) css (style 태그)
   - style 태그는 head 사이에 삽입
2. title : 문서의 제목
3. body 브라우저에 표시되는 내용들
